## Why

提升小朋友在練習注音過程中的成就感與持續動力。透過擊敗魔王後獲得「貼紙」的獎勵機制，建立一個長期的收集目標，讓練習過程更像一場有趣的冒險。

## What Changes

- **新增獎勵機制**：擊敗大魔王（第 6 關）後，玩家將獲得一張獨特的「童趣風格貼紙」。
- **設計 10 款貼紙**：包含奧特曼、寶可夢、迪士尼、馬力歐等主題的 Q 版設計（非 Emoji）。
- **新增「貼紙收集冊」**：在主畫面新增一個按鈕，點擊後可開啟收集冊，查看已獲得與尚未獲得的貼紙。
- **持久化儲存**：使用 `localStorage` 記錄已收集的貼紙，確保重新整理頁面後紀錄不會消失。
- **優化獎勵動畫**：更新擊敗魔王後的彈窗，增加貼紙展示與飛入收集冊的動態效果。

## Capabilities

### New Capabilities
- `sticker-engine`: 負責貼紙的隨機抽取、儲存邏輯與狀態管理。
- `sticker-book-ui`: 實作貼紙收集冊的介面，包含展示牆、分頁或進度條。
- `sticker-assets`: 定義與渲染 10 款貼紙的視覺資源（可能是 SVG 或 CSS Art）。

### Modified Capabilities
- `game-reward-flow`: 修改現有的魔王擊敗邏輯，從單純顯示「糖果」改為觸發「獲得貼紙」流程。

## Impact

- **index.html**: 需要新增收集冊的 HTML 結構、CSS 樣式以及相關的 JavaScript 獎勵邏輯。
- **localStorage**: 將使用 `localStorage` 儲存已獲得貼紙的 ID 陣列。
